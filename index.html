Clínica Médica Dr. Enias

HTML
Css
Site cujo desafio era ser responsivo utilizando apenas HTML e CSS para uma Clínica Médica Dr. Enias. O projeto destaca-se pela simplicidade e eficiência, com um design limpo e profissional que reflete o cuidado e a confiabilidade da clínica. As páginas incluem informações sobre os serviços oferecidos, a equipe médica, horários de atendimento e um formulário de contato.

Era do Ouro: E-commerce

HTML
Css
JavaScript
Site Era do Ouro, um e-commerce desenvolvido com HTML, CSS e Javascript, focado na venda de produtos de alta qualidade. O design foi pensado em ser uma verdadeira máquina do tempo a Era Colonial do Brasil, destacando os produtos e itens vendidos na época. Inclui páginas de catálogo, detalhes de produtos e um carrinho de compras totalmente funcional. O layout é totalmente responsivo, garantindo uma experiência de compra fluida em desktops.

Previsão de Jogos

Python
Este site utiliza técnicas de machine learning para analisar e prever resultados de partidas de League of Legends (LoL). Por meio de dados históricos de equipes, desempenho individual dos jogadores e estatísticas de jogos anteriores, o sistema identifica padrões e gera previsões detalhadas e precisas.
